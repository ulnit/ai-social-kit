# 📱 AI Social Media Kit — Cross-Platform Auto Posting

**AI generates tailored posts for Twitter/X, LinkedIn, Reddit, Threads, and Dev.to from product descriptions.**

## What Is This?

A Python engine that takes a product description and auto-generates platform-optimized social posts. Different templates, tone, and format for each platform. Perfect for product launches, daily promotion, and multi-channel growth.

## Quick Start

```bash
# Generate posts for all platforms
python3 engine.py --product "AI Video Factory"

# Single platform
python3 engine.py --product "AI API Gateway" --platform reddit

# Save to file
python3 engine.py --product "AI Video Factory" --output posts.json
```
